# Modals Component Library

Reusable modal components for the Nekzus terminal dashboard.

## Components

### Modal (Base)
Base modal wrapper with accessibility features.

### ConfirmationModal
Delete/action confirmation dialog with danger mode.

### EditRouteModal
Complete add/edit route form with validation.

## Usage

```javascript
import { Modal, ConfirmationModal, EditRouteModal } from './components/modals';
```

## Full Documentation

See `/ROUTES_FOUNDATION.md` for complete API reference and examples.

## Files

- `Modal.jsx` - Base modal component

- `ConfirmationModal.jsx` - Confirmation dialog

- `EditRouteModal.jsx` - Route form modal
